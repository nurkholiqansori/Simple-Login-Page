# Simple Login System


Build with:

- Express.js
- Node.js
- MongoDB (Mongoose)
- Passport.js
- EJS Templates
- Bootstrap

## Features

- Login
- Registration
- Logout
- Dashboard

## Installation

- Install the dependencies

    ```bash
    npm install
    ```

- Run the server

    ```bash
    npm run start
    ```
The server will be running on port 3000.
